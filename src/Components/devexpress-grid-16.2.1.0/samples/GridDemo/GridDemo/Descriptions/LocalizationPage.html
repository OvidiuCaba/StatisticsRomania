Localization

This demo illustrates how resources can be localized for a specific culture to display an application with the GridControl in a different language.

In this demo, you can select a culture to be applied to the grid by tapping the corresponding button on the right. When the language is changed, the grid changes the following.

  • Column captions. Each column's caption is set to a corresponding resource value.
  • Number and date formatting. The grid columns automatically display numbers and dates based on the applied culture.
  • Text of the grid elements. The control automatically retrieves the required strings from the provided localization resources.
