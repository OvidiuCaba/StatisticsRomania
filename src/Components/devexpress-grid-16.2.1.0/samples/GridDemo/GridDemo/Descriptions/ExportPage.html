Export

This demo illustrates Excel export capabilities of the GridControl. Our data export engine can be used for subsequent analysis of grid data in an XLSX, XLX or CSV document.

The following data shaping options are fully supported for all Excel export operations:

  • Data Grouping – with the ability to collapse/expand groups within a worksheet.
  • Data Sorting and Filtering – allowing end-users to display relevant data in the desired order.
  • Total and Group Summaries – with the ability to modify/change formulas.

After you export the grid data, tap the Open button to open an output file using an application for working with Excel files installed on your device.
